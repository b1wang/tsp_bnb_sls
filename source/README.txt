To execute the program use

name_of_tsp_file.out is the file generated from the generate_travelling_salesman.py

BnB DFS:
	python bnb_dfs.py --tsp_file name_of_tsp_file.out
	or
	python3 bnb_dfs.py --tsp_file name_of_tsp_file.out

SLS:
	python sls.py --tsp_file name_of_tsp_file.out 
	or
	python3 sls.py --tsp_file name_of_tsp_file.out
